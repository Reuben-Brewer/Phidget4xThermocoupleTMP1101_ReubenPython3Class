Phidget4xThermocoupleTMP1101_ReubenPython3Class

Control class (including ability to hook to Tkinter GUI) to interface with 4x thermocouple-reader TMP1101_1 from Phidgets.

https://www.phidgets.com/?prodid=1215&pcid=12

TMP1101_1: This thermocouple interface has four inputs and a maximum data rate of 50 Hz.

########################### Python module installation instructions, all OS's

Phidget4xThermocoupleTMP1101_ReubenPython3Class, ListOfModuleDependencies_All:['CSVdataLogger_ReubenPython3Class', 'EntryListWithBlinking_ReubenPython2and3Class', 'future.builtins', 'GetCPUandMemoryUsageOfProcessByPID_ReubenPython3Class', 'keyboard', 'LowPassFilterForDictsOfLists_ReubenPython2and3Class', 'MyPlotterPureTkinterStandAloneProcess_ReubenPython2and3Class', 'MyPrint_ReubenPython2and3Class', 'numpy', 'pexpect', 'Phidget22', 'psutil', 'pyautogui']

https://pypi.org/project/Phidget22/#files

To install the Python module using pip:

pip install Phidget22       (with "sudo" if on Linux/Raspberry Pi)

To install the Python module from the downloaded .tar.gz file, enter downloaded folder and type "python setup.py install"
